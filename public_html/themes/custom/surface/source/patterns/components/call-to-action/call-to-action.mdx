# Call to action

The Call to action component is made up of two parts, a title and a button. Both parts are grouped in a container with background color. This is ideal for calling out a page or a site so it stands out.

### Default Call to action

<Story of={CTAStories.CTA} />

## Call to action technical specs

* `modifier` makes it possible to pass a CSS modifier class.
* `title` gives the component a title.
* `url` is the link for the CTA button.
* `url_title` is the CTA button's label.
