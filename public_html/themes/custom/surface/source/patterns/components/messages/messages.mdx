# Messages

## Messages technical specs

* `type` is the type of message being displayed. See below for examples.
* `heading` is a visually hidden title for a11y purposes.
* `messages[]` is an array of message of any type which can be displayed as a single message or multiple ones. The messages auto-disappear after a few seconds.
