# Accordion

Most commonly used for FAQ format of content, the Accordion is a great way to present content while saving space on a page.

### Default Accordion

<Story of={AccordionStories.Accordion} />

## Accordion technical specs

* `modifier` makes it possible to pass a CSS modifier class.
* `id` is a unique numeric value unique to each item.
* `title` is the visible piece of content of question in the FAQ, which also acts as a toggle to display the answer.
* `text` is the answer to the question only visible when the question is toggled on.
