# Accordion tabs

The Accordion tabs component provides the ability to add multiple pieces of content separated into tabs. This is a great way to save space on a page while still providing a lot of content in a well-organized manner.

## Accordion tabs technical specs

* `modifier` makes it possible to pass a CSS modifier class.
* `items[]` is an array in which each item becomes a separate tab with its own content of any kind (text, images, videos, etc.).
  * `entity` is the key for each item/tab.
  * `id` is a unique numeric value unique to each item.
  * `field_sf_title` is the title of the tab.
  * `field_sf_wysiwyg` is the text field for each tab's content.
